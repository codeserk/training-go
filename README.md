# Training: GO

This repository contains a simple GO project that is created while I have been learning how to write GO.

## Features

- [X] Clean architecture demonstration (http, use cases, repository)
- [X] API using Swagger

## Installation

### Instal GO dependencies

```bash
go mod tidy
```

## Usage

```bash
go run .
```
